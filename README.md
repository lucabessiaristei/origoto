# Origoto - FOLD Layer Editor

A vibecoded visual tool for cheating face stacking orders in `.fold` files.

## Features
- **Navigation**: Centered zoom and spacebar/middle-click panning.
- **Quick Order Tool**:
    - **Click**: Send to back.
    - **Alt + Click**: Bring to front.
    - **Right Click**: Select face.
- **Sidebar**: Drag-and-drop manual reordering of individual faces.
- **History**: Full undo/redo support.
- **Export**: Saves valid FOLD JSON with updated `faces_layer` data.
